# 🏥 Hospital Inventory Management System

A Python-based Hospital Inventory Management System designed to manage medicine stock efficiently through a simple command-line interface. This project demonstrates the use of Python fundamentals, including dictionaries, functions, loops, and control flow to perform inventory management operations.

---

## 📋 Project Overview

Managing hospital medicine inventory is an important task to ensure that essential medicines are always available. This application allows users to add, search, update, and remove medicines while maintaining an organized inventory system.

The project is completely menu-driven and provides an easy-to-use interface for inventory management.

---

## ✨ Features

✅ View complete inventory list

✅ Search medicines by name

✅ Add new medicines to inventory

✅ Increase or decrease stock quantities

✅ Remove medicines from inventory

✅ Interactive menu-driven interface

✅ Dictionary-based data management
